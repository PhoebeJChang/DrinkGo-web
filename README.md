# Drink Go web term project
DrinkGo飲料訂購網站(輔大周圍)

## Introduction
這是一個關於輔大附近飲料店資訊的網站，大家可以透過這個網站查詢學校附近的飲料店，
可以先定位自己的所在位置，選擇自己喜歡的或是距離較近的飲料店後，
再用不同的分類來選擇，像是口味或是價格，也會有當季的特價或是推薦的飲品，
調整甜度冰塊和要加的料後，即可放入購物車。
</br>
*各商品無法真正訂購，我們設定下單成功後十分鐘會自動完成訂單。
